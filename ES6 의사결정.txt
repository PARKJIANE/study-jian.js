JS 의사결정
부울: boolean
1. if문 : 'if' 문은 부울 표현식과 하나 이상의 문으로 구성됩니다.
2. if...else : 'if' 문 다음에 선택적 'else' 문이 올 수 있으며, 이는 부울 표현식이 거짓일 때 실행됩니다.
3. else.. if 사다리/중첩 if 문 : else…if 사다리는 여러 조건을 테스트하는 데 유용합니다. 다음은 동일한 구문입니다.
4. switch...case 문 : switch 문은 표현식을 평가하고 표현식의 값을 케이스 절과 일치시키고 해당 케이스와 관련된 명령문을 실행합니다.
