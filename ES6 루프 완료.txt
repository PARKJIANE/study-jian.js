ES6 루프

한정 루프 : 반복 횟수가 한정/고정된 루프를 한정 루프 라고 합니다 . 'for 루프'는 한정 루프 의 구현입니다 .
1. for 루프 : for 루프는 지정된 횟수만큼 코드 블록을 실행합니다.
2. for...in 루프 : for...in 루프는 객체의 속성을 반복하는 데 사용됩니다.
3. for...of 루프 : for…of 루프는 object literals 대신 iterables을 반복하는 데 사용됩니다.

무한 루프 : 무한 루프는 루프의 반복 횟수가 불확실하거나 알 수 없는 경우에 사용됩니다.
1. while 루프 : while 루프는 지정된 조건이 true로 평가될 때마다 명령을 실행합니다.
2. do...while 루프 : do...while 루프는 루프가 처음 실행될 때 do...while 루프가 조건을 평가하지 않는다는 점을 제외하고는 while 루프와 유사합니다.

루프 제어문 
1. break 문 : break 문은 구문에서 제어를 가져오는 데 사용됩니다.
2. continue 문 : 계속 문은 현재 반복의 후속 문을 건너뛰고 제어를 루프의 시작 부분으로 되돌립니다.

레이블을 사용하여 흐름 제어
1. break있는 레이블 : 레이블을 break와 함께 사용하여 흐름을 더 정확하게 제어할 수 있습니다.
2. continue있는 레이블 : 'continue' 또는 'break' 문과 해당 레이블 이름 사이에는 줄 바꿈이 허용되지 않습니다.
